# ACP Trademark and Certification Usage

This document describes the permitted use of the **Active Cyber Program (ACP)** name, certification, and trust label.

The purpose of this document is to protect the integrity and credibility of the ACP framework and certification program.

---

# ACP Framework

The **Active Cyber Program (ACP)** framework is published to promote transparency and support organizations in improving their cybersecurity programs.

Organizations and individuals may:

* reference the ACP framework
* use the framework for internal cybersecurity assessments
* use the framework for cybersecurity improvement initiatives
* study the framework for educational purposes

However, the framework must not be misrepresented as an official certification without authorization.

---

# ACP Certification

The **Active Cyber Program (ACP) Certification** may only be issued by:

**Wechsler Information Solution**

Organizations or individuals may **not issue ACP certifications** unless explicitly authorized by Wechsler Information Solution.

Unauthorized use of the ACP certification or representation as an ACP certification authority is not permitted.

---

# ACP Trust Label

The **ACP Trust Label** may only be used by organizations that have received a valid **Active Cyber Program Certification**.

The trust label may be used in:

* corporate websites
* marketing materials
* security documentation
* supplier security responses

The trust label must not be:

* modified
* altered in design
* used after certification expiration
* used in a misleading manner

---

# Certification Validity

ACP certifications are valid for **24 months** unless otherwise stated.

Organizations must undergo reassessment to maintain certification and continue using the ACP Trust Label.

---
