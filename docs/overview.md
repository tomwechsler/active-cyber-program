# Active Cyber Program (ACP) – Framework Overview

The **Active Cyber Program (ACP)** is a cybersecurity assessment and certification framework designed to evaluate whether an organization operates an **active and effective cybersecurity program**.

The framework focuses on verifying that cybersecurity is **not only implemented but actively managed** across governance, operational processes, and technical security controls.

ACP provides organizations with a structured method to understand, evaluate, and improve their cybersecurity posture.

---

# Purpose of the ACP Framework

Many organizations implement individual security technologies or policies but lack a structured and actively managed cybersecurity program.

The ACP framework helps organizations:

* establish a structured cybersecurity program
* evaluate cybersecurity maturity
* identify security improvement opportunities
* demonstrate cybersecurity commitment to customers and partners

The framework is designed to support organizations in building **sustainable and operational cybersecurity practices**.

# Core Elements of ACP

The Active Cyber Program framework consists of several key components.

## ACP Principles

The ACP principles define the fundamental philosophy behind the framework.

They emphasize that cybersecurity must be:

* business-driven
* actively managed
* risk-based
* continuously improved

## Control Domains

The framework defines **ten cybersecurity control domains** covering governance, operational processes, and technical protection mechanisms.

These domains provide a structured way to evaluate cybersecurity capabilities across an organization.

See:

`framework/control-domains.md`

---

## Maturity Model

ACP uses a maturity model to evaluate how effectively cybersecurity practices are implemented.

The maturity model includes five levels:

1. Initial
2. Managed
3. Defined
4. Measured
5. Optimized

See:

`framework/maturity-levels.md`

Organizations that successfully meet the requirements of the ACP framework may receive the:

The certification confirms that the organization operates an **active cybersecurity program**.

# Target Organizations

The ACP framework is designed for organizations of all sizes, including:

* small and medium-sized enterprises (SMEs)
* technology companies
* service providers
* public sector organizations
* suppliers in security-sensitive industries

The framework is designed to be **practical, scalable, and applicable across industries**.

---

# Continuous Development

Cybersecurity challenges evolve continuously.

The ACP framework will therefore evolve over time based on:

* new cybersecurity threats
* operational experience from assessments
* improvements in security practices
* feedback from organizations

New framework versions may be published periodically.

The **Active Cyber Program (ACP)** provides organizations with a structured and practical approach to evaluate and improve their cybersecurity programs.
