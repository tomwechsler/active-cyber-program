# ACP Certification

The **Active Cyber Program (ACP) Certification** confirms that an organization operates an active and structured cybersecurity program aligned with the ACP framework.

Certification is granted after a successful ACP assessment demonstrating that cybersecurity governance, operational processes, and technical protections are actively implemented and managed.

---

# Certification Principles

The ACP certification is based on the following principles:

* transparency
* evidence-based evaluation
* practical applicability
* scalability for organizations of all sizes
* continuous improvement

Certification does not guarantee the absence of security incidents. It confirms that an organization operates an **active cybersecurity management program**.

---

# Assessment Requirements

To qualify for ACP Certification, an organization must undergo a formal ACP assessment.

The assessment evaluates:

* ACP control domains
* implemented security controls
* operational security practices
* supporting evidence

Assessment activities may include:

* interviews with responsible personnel
* review of security documentation
* technical validation
* review of operational processes

---

# Certification Scoring Model

Each control in the ACP checklist is evaluated using the following scoring model.

| Score | Description                   |
| ----- | ----------------------------- |
| 0     | Control not implemented       |
| 1     | Control partially implemented |
| 2     | Control implemented           |
| 3     | Control actively managed      |

Each domain receives an average score based on its controls.

---

# Minimum Certification Requirements

An organization may receive **ACP Certification** if the following conditions are met:

* Overall average score ≥ **2.0**
* No critical domain score below **1.5**
* Evidence supporting the implementation of controls is provided
* Cybersecurity governance is established

The assessor may define certain domains as **critical domains**, depending on the organization’s risk profile.

---

# Certification Result

The ACP assessment produces the following deliverables:

* ACP Assessment Report
* Domain maturity overview
* Identified improvement areas
* ACP Certification decision

Organizations that meet the certification criteria receive the **Active Cyber Program (ACP) Certification**.

---

# Certification Validity

ACP Certification is valid for:

**24 months**

Cybersecurity environments change continuously; therefore periodic reassessment is required to maintain certification.

---

# Reassessment

Organizations may renew their ACP Certification by completing a reassessment before the expiration date.

Reassessment verifies that:

* security controls remain active
* improvements have been implemented
* new risks are addressed

---

# Use of ACP Certification

Organizations that receive ACP Certification may state that they operate an **Active Cyber Program (ACP)**.

They may reference the certification in:

* corporate security statements
* customer security questionnaires
* supplier security documentation
* security assurance documentation

The certification must not be used in a misleading way suggesting absolute security or risk elimination.
