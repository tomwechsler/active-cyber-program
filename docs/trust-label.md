# ACP Trust Label

The **ACP Trust Label** allows organizations to publicly demonstrate that they operate an **Active Cyber Program (ACP)**.

The label communicates that the organization has successfully completed an ACP assessment and meets the requirements of the ACP framework.

---

# Label Purpose

The ACP Trust Label helps organizations:

* demonstrate cybersecurity maturity
* build trust with customers and partners
* communicate security posture
* support supplier security requirements

---

# Label Variants

## ACP Certified

Indicates that an organization has successfully achieved **Active Cyber Program Certification**.

## ACP Verified

Indicates that an organization has completed an ACP assessment but may still be implementing improvement actions.

## ACP Partner

Indicates an organization authorized to perform ACP assessments.

---

# Label Usage Rules

Organizations may display the ACP Trust Label on:

* corporate websites
* marketing material
* security documentation
* customer security responses

The label must not be modified or used in a misleading way.

---

# Certification Validity

The ACP Trust Label may only be used during the validity period of the ACP Certification.

Once certification expires, the label must be removed from public materials.
